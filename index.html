HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The alt attribute contains information for screen readers and other assistive technology. It also provides better search engine rankings.
  • The h1 tag should only appear once per page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • When selection an element, we can write h1, a, *, but when selecting a class it must have a . (ie. .class), id must have #

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main will switch to main branch
  • git addd -A commits all files in the working branch
  • git commit -m "message": commits changes, -m associates a message with our commit, make it descriptive of the changes
  • git pull origin main: pulls in the base branch
  • git push origin branch/name: pushes up all of our changes
